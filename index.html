Projet ESP32 avec Firebase

Definitions:

  • ESP32

    L'ESP32 est une puce microcontrôleur à 32 bits basée sur le microcontrôleur Tensilica Xtensa LX6. Il est conçu pour être utilisé dans des applications IoT (Internet des objets) et embarque un Wi-Fi et un Bluetooth.

  • DHT11

    Le capteur de température et d'humidité DHT11 est un capteur numérique qui peut mesurer la température et l'humidité ambiante. Il est basé sur le capteur de température et d'humidité DHT.

  • Firebase

    Firebase est une plateforme de développement d'applications mobiles et Web qui fournit des services d'hébergement d'applications, de base de données, de messagerie et de stockage de fichiers.

Matériel nécessaire:

  • Carte microcontrôleur ESP32
  • Capteur de température et d'humidité dht11
  • Câbles de raccordement
  • Connexion Internet

Étape 1: Créer un projet Firebase

  1. Allez sur le site web de Firebase (https://firebase.google.com) et connectez-vous à votre compte Google.
  2. Cliquez sur "Ajouter un projet" et donnez-lui un nom. Sélectionnez également votre pays/région et cliquez sur "Créer un projet".
  3. Dans la page de présentation du projet, cliquez sur "Ajouter Firebase à votre application web".
  4. Dans la boîte de dialogue qui s'ouvre, donnez un nom à votre application et copiez les informations de configuration (apiKey, authDomain, databaseURL, projectId, storageBucket, messagingSenderId, appId) dans un éditeur de texte.

Étape 2: Créer une base de données Firebase

  1. Accédez à la console Firebase, sélectionnez votre projet, puis cliquez sur "Realtime Database".
  2. Cliquez sur "Créer une base de données" et sélectionnez le mode "Testez le mode".
  3. Définissez les règles d'accès à votre base de données en modifiant la valeur "false" à "true" dans la règle "read" et "write" : 
    4. 
          {
            "rules": {
              ".read": true,
              ".write": true
            }
          }

Étape 3: Connecter le DHT11 à l'ESP32

  1. Connectez le fil rouge (+) du capteur DHT11 au pin 5V de l'ESP32.
  2. Connectez le fil noir (-) du capteur DHT11 au pin GND de l'ESP32.
  3. Connectez le fil jaune (signal) du capteur DHT11 au pin D14 de l'ESP32.

Étape 4: Envoyer les données de température et d'humidité à Firebase

Utilisez le code fourni dans votre environnement de développement préféré pour envoyer les données de température et d'humidité à Firebase.

ESP32 - Microcontroller

L'ESP32 est un microcontrôleur Wi-Fi de la société Espressif Systems. Il est souvent utilisé pour les projets d'Internet des objets (IoT) et les projets de développement de prototypes.

Caractéristiques de l'ESP32 :

  • Processeur à deux cœurs cadencé à 240 MHz
  • Wi-Fi intégré (802.11 b/g/n)
  • Bluetooth intégré (jusqu'à Bluetooth 5.0)
  • Jusqu'à 34 broches d'entrée/sortie (GPIO)
  • Jusqu'à 18 canaux d'entrée analogique
  • 3 UART, 3 SPI, 2 I2C, 2 I2S, PWM, RTC, capteurs de température, capteurs de toucher
  • Mémoire flash intégrée de 4 Mo

Exemples de projets IoT ESP32

Voici cinq exemples de projets IoT basés sur ESP32 :

  1. Système de surveillance de la température pour maison intelligente

    Ce projet utilise un ESP32 et un capteur de température pour surveiller la température d'une maison et envoyer des notifications sur le téléphone de l'utilisateur si la température dépasse un certain seuil. Le système peut également être contrôlé à distance via une interface web.

    Pour réaliser ce projet, il faut brancher le capteur de température sur l'ESP32 et programmer le microcontrôleur pour lire les données du capteur et les envoyer à un serveur web. Le serveur web peut ensuite envoyer des notifications sur le téléphone de l'utilisateur si la température dépasse le seuil prédéfini. En utilisant une interface web, l'utilisateur peut également surveiller la température de sa maison à distance et contrôler les paramètres du système de surveillance.

  2. Système d'arrosage automatique pour plantes

    Ce projet utilise un ESP32 et des capteurs d'humidité pour surveiller le niveau d'humidité des plantes et les arroser automatiquement si nécessaire. Le système peut être configuré pour arroser différentes plantes à différents moments et intervalles.

    Pour réaliser ce projet, il faut brancher les capteurs d'humidité sur l'ESP32 et programmer le microcontrôleur pour surveiller les données des capteurs et contrôler l'arrosage des plantes. Le système peut également être équipé d'une interface web pour permettre à l'utilisateur de surveiller l'état des plantes et de contrôler les paramètres du système d'arrosage.

  3. Système de suivi de véhicule

    Ce projet utilise un ESP32 et un module GPS pour suivre la position d'un véhicule en temps réel. Le système peut envoyer les données de localisation à un serveur web et afficher la position du véhicule sur une carte.

    Pour réaliser ce projet, il faut brancher le module GPS sur l'ESP32 et programmer le microcontrôleur pour lire les données de localisation et les envoyer à un serveur web. Le serveur web peut ensuite afficher la position du véhicule sur une carte et permettre à l'utilisateur de surveiller le trajet du véhicule en temps réel.

  4. Système de verrouillage de porte intelligent

    Ce projet utilise un ESP32 et un moteur servo pour contrôler un verrou de porte. Le système peut être configuré pour verrouiller et déverrouiller la porte automatiquement en fonction de l'heure ou des commandes envoyées depuis une application mobile ou une interface web.

    Pour réaliser ce projet, il faut connecter le moteur servo au circuit de l'ESP32 et programmer le microcontrôleur pour contrôler le mouvement du servo en fonction des commandes reçues. Le système peut être équipé d'une application mobile ou d'une interface web pour permettre à l'utilisateur de verrouiller ou déverrouiller la porte à distance.

  5. Système de contrôle de la qualité de l'air

    Ce projet utilise un ESP32 et des capteurs de qualité de l'air pour surveiller les niveaux de polluants dans l'air et envoyer des alertes si les niveaux dépassent un certain seuil. Le système peut également être équipé d'une interface web pour permettre à l'utilisateur de surveiller les niveaux de polluants en temps réel et de contrôler les paramètres du système.

    Pour réaliser ce projet, il faut brancher les capteurs de qualité de l'air sur l'ESP32 et programmer le microcontrôleur pour lire les données des capteurs et les envoyer à un serveur web. Le serveur web peut ensuite envoyer des alertes sur le téléphone de l'utilisateur si les niveaux de polluants dépassent le seuil prédéfini. En utilisant une interface web, l'utilisateur peut également surveiller les niveaux de polluants en temps réel et contrôler les paramètres du système de surveillance.
